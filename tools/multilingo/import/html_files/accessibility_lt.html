Prieinamumas

Europeana.eu pobūdį ir turinį buvo stengtasi sukurti taip, kad vartotojams, naudojant tam tikras priemones bei įrenginius, jis būtų prieinamas visur. Mes stengiamės atitikti WC3 Tinklalapio turinio prieigos rekomendacijų Versiją 1.0 (WC3 Web Content Accessibility Guidelines Version 1.0).

Į vartotoją orientuota mūsų kūrėjų komanda dažniausiai atlieka naudojimosi galimybių tyrimus, testuojant Europeana.eu navigaciją, srautus, nuoseklumą. Be to, mes pateikiame versiją – poprototipiniam etapui – akliesiems.

Kai kurie mūsų tinklalapio prieigos bruožai:

  • Prasmingos puslapių antraštės, padedančios orientuotis
  • Beveik visi vaizdai turi prasmingą ekvivalentišką tekstą
  • Rėmeliai naudojami minimaliai
  • Naudojami įvairių dydžių šriftai
  • Naudojama Cascading Style Sheets (CSS), siekiant atskirti turinį nuo pristatymo
  • Pranešama vartotojui apie tai, kad atveriamas naujas langas
  • (Pagrindinės) funkcijos atliekamos nenaudojant JavaScript

Prašome mums siųsti, naudojant grįžtamojo ryšio formą, savo komentarus ar klausimus dėl Europeana.eu prieigos.
