Dostupnost

Vzhled a obsah Europeana.eu byly navrženy tak, aby byly všeobecne prístupné uživatelum a uživatelským agentum / zarízením. Snažíme se vyhovet WC3 Web Content Accessibility Guidelines Version 1.0.

Náš uživatelský centrální tým casto provádí testování použitelnosti navádení, toku, oznacení a konzistence Europeana.eu. Následovne budeme vyrábet verzi - po této fázi prototypu - pro nevidomé jednotlivce.

Nekteré z funkcí prístupnosti našich stránek obsahují:

  • Smysluplnou titulní stránku napomáhající s orientací
  • MVetšina obrázku má jeden smysluplný alt-text (textový ekvivalent)
  • Minimální použití rámcu
  • Použití velikosti písma
  • Použití kaskádových formátovacích stylu (CSS) pro oddelení obsahu od prezentace
  • Oznámení uživateli, když se odkaz otevre v novém okne
  • (Hlavní) funkce použitelné bez JavaScriptu

Zašlete nám prosím jakékoli dotazy nebo námety ohledne dostupnosti Europeana.eu spolu s formulárem dotazu a námetu..
