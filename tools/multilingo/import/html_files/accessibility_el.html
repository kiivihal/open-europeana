Προσβασιμότητα

Τα χαρακτηριστικά και το περιεχόμενο του Europeana.eu έχουν σχεδιαστεί για να καθίστανται προσβάσιμα παγκοσμίως από χρήστες και παράγοντες/συσκευές χρηστών. Πασχίζουμε να ανταποκρινόμασμε στις Κατευθυντήριες Οδηγίες Προσβασιμότητας σε Περιεχόμενο Ιστού (WC3 Web Content Accessibility Guidelines Version 1.0).

Η Ομάδα Σχεδιασμού με Επίκεντρο τον Χρήστη διενεργεί συχνά δοκιμές χρηστικότητας της πλοήγησης, ροής, ετικετών και συνέπειας του Europeana.eu. Περαιτέρω θα αναπτύξουμε μιαν έκδοση – μετά τη φάση του πρότυπου προγράμματος – για τυφλούς.

Ορισμένα από τα χαρακτηριστικά προσβασιμότητας του δικτυακού τόπου μας περιλαμβάνουν:

  • νοηματικά σημαντικές σελίδες τίτλου για να βοηθούν στον προσανατολισμό
  • οι περισσότερες εικόνες έχουν νοηματικά σημαντικό κείμενο (ισοδύναμο κειμένου)
  • ελάχιστη χρήση πλαισίων
  • χρήση γραμματοσειρών που αλλάζουν σχήμα
  • χρήση Cascading Style Sheets (CSS: Στοιβαγμένα Φύλλα Στυλ) για να διαχωρίζεται το περιεχόμενο από την παρουσίαση.
  • ειδοποίηση του χρήστη όταν ανοίγει μια σύνδεση σε νέο παράθυρο.
  • οι (κύριες) λειτουργίες χρησιμοποιούνται χωρίς τη JavaScript.

Παρακαλούμε να μας αποστείλετε ανάδραση ή ερωτήσεις για την προσβασιμότητα του Europeana.eu, με τη φόρμα ανάδρασης.
