Zugänglichkeit

Inhalte und Angebote von Europeana.eu wurden so gestaltet, dass sie universell durch Nutzer und Geräte zugänglich sind. Wir bemühen uns, die W3C Web Content Accessibility-Leitlinien Version 1.0 zu erfüllen

Unser Team für nutzerorientiertes Design führt häufig Nutzertests für die Navigation, die verwendeten Beschriftungen und die Konsistenz der Europeana.eu durch. Wir werden außerdem – nach der Prototyp-Phase – eine Version für Blinde erstellen.

Einige Merkmale der guten Zugänglichkeit unserer Website umfassen:

  • Sinnvolle Seitenbezeichnungen, um die Orientierung zu erleichtern
  • Ein sinnvolles Textäquivalent für die meisten Bilder
  • Minimale Verwendung von Frames
  • Verwendung von skalierbaren Fonts
  • Verwendung von Cascading Style Sheets (CSS), um Inhalt von Präsentation zu trennen
  • Information des Nutzers, wenn ein Link ein neues Fenster öffnet
  • Die (wesentlichen) Funktionalitäten sind ohne JavaScript nutzbar
