Πολιτική γλώσσας

Το Europeana.eu ακολουθεί όμοιες κατευθυντήριες οδηγίες πολιτικής με το Europa (την ηλεκτρονική πύλη της Ευρωπαϊκής Ένωσης).

Το δυνατόν συντομότερο, στόχος είναι να παρέχονται στο κοινό οι πληροφορίες που αναζητά στη γλώσσα του. Όμως, εξαιτίας περιορισμών σε πόρους, αυτό δεν είναι πάντοτε εφικτό.

Επομένως οι ακόλουθες κατευθυντήριες οδηγίες ισχύουν στο Europa και θα ισχύουν στο Europeana.eu:

  • Όλες οι σελίδες ανωτέρου επιπέδου (*) στις γλώσσες όλων των εταίρων
  • Δίνεται προτεραιότητα στην ταχύτητα δημοσίευσης έναντι της διαθεσιμότητας της μετάφρασης
  • Επιτρέπεται περιορισμένη «ανάμειξη γλωσσών» (**), δηλαδή: όπου χρειάζεται να εμφανιστεί ένα προειδοποιητικό σήμα.

(*) π.χ.: διεπαφές πλοήγησης, έρευνας, αναζήτησης και εμφάνισης

(**) Ανάμειξη γλωσσών = περισσότερες από μια γλώσσες σε μια σελίδα

Στην πράξη αυτό σημαίνει τα ακόλουθα:

  1. Τα εργαλεία διαχείρισης της μετάφρασης θα αναπτύσσονται και θα διατηρούνται από το Europeana.eu.
  2. Η ολοκλήρωση των μεταφράσεων θα αποτελεί ευθύνη κάθε καθορισθέντος ιδρύματος του εθελοντή εταίρου.
  3. Το Europeana.eu θα ειδοποιεί κάθε καθορισθέν ίδρυμα του εθελοντή εταίρου, όταν απαιτείται ανανέωση και δεν θα το πιέχει για την ανανέωση αυτή.
  4. Πληροφορίες που είναι επείγουσες ή έχουν μικρή διάρκεια ζωής θα παρατίθενται μόνο στα Αγγλικά, π.χ. εξειδικευμένα μηνύματα λαθών.
  5. Όλα τα ερωτηματολόγια για τον παρόντα δικτυακό τόπο θα παράγονται μόνο σε όλες τις εφικτές γλώσσες.
  6. Στην αρχή η Προώθηση Μηχανής Αναζήτησης θα εκτελείται στα Αγγλικά.
  7. Το τμήμα της Διαχείρισης και οι τεχνικές πληροφορίες θα διατηρούνται μόνο στα Αγγλικά, ως γλώσσα εργασίας του Europeana.eu.
  8. Αν όμως ένα ίδρυμα επιθυμεί να μεταφράζει και να διατηρεί οποιοδήποτε από τα παραπάνω με δικούς του πόρους, το Europeana.eu θα είναι ικανοποιημένο να διευκολύνει την εμφάνισή του.
