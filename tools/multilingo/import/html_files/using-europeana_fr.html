Utiliser Europeana

Bienvenue sur Europeana. Vous trouverez ici une brève présentation de quelques fonctionnalités offertes par le site.

Rechercher dans Europeana

Vous pouvez, sur Europeana, consulter des millions d’objets numériques provenant des musées, galeries, archives, bibliothèques et institutions audio-visuelles européennes.

Effectuer une recherche sur Europeana est simple. Posez-vous simplement les questions suivantes : qui, quoi, où, quand et interrogez le moteur de recherche de la façon suivante :

  • Qui: par nom d’acteurs, auteurs, architectes, artistes, chorégraphes, compositeurs, chefs d’orchestre, danseurs, metteurs en scène, musiciens ou photographes.
  • Quoi: par mots du titre pour les livres, poèmes, journaux, peintures, photographies, films ou programmes de télévision.
  • Où: par nom de villes ou de pays, européens ou non-européens.
  • Quand: par date (ex.: 1945) comme l’année de votre naissance ou une date célèbre ou alors une période (ex : époque romaine ou moyen-âge)

La recherche avancée vous permet d’affiner votre requête dans les mots du titre, dans la recherche par auteur (ex. : auteurs, artistes, musiciens etc…) ou dans la recherche par date (ex. : 1945). Vous pouvez aussi combiner plusieurs critères de recherche (ex. : titre: Hamlet et date: 1983)

Résultats dans Europeana

Les résultats de votre recherche sont regroupés par type de document: texte, image, vidéo et son :

  • Texte : livres, lettres, archives, thèses, poèmes, journaux, articles, fac-similés, manuscrits et partitions.
  • Image : peintures, dessins, estampe, photographies, images, objets de musées, cartes, dessins, schémas, plans et partitions musicales.
  • Vidéo : films, journaux télévisés et programmes de télévision.
  • Son : musique et enregistrements sonores sur cylindres, cassettes, disques et émissions de radio.

Vous pouvez choisir d’afficher les résultats sous forme de listes ou de tableaux .

Pour afficher les détails d’un objet, cliquer sur l’objet dans la liste des résultats.

Si vous obtenez un nombre trop important de résultats, vous avez la possibilité d’affiner encore votre recherche en sélectionnant par langue, pays, date, source ou par type à partir du menu qui se trouve à gauche de l’écran des résultats.

Pour lire le texte, visualiser l’image, regarder une vidéo ou écouter un document sonore, cliquer sur l’image de l’objet qui s’affiche sur la page détails de l’objet. Vous avez aussi la possibilité d’aller sur le lien rattaché à l’objet pour le visualiser dans son contexte d’origine. Ce lien vous redirigera sur le site de l’institution qui donne accès à cet objet depuis Europeana.

A gauche de la page de l’objet sélectionné, Europeana affiche les documents qui lui sont liés.

MonEuropeana

Vous n’avez pas besoin de vous enregistrer pour utiliser Europeana. Néanmoins, l’enregistrement permet aux utilisateurs d’accéder à un plus nombre de fonctionnalités telles que :

  • Sauvegarder sa recherche: possibilité de garder la recherche que vous venez de faire sur un sujet spécifique dans Europeana pour la relancer ultérieurement.
  • Ajouter une étiquette: ajouter vos propres étiquettes sur vos objets préférés.
  • Sauvegarder MonEuropeana: sauvegarder vos objets sélectionnés pour les consulter ultérieurement.
  • Partager avec un ami: partager avec vos amis les objets qui vous intéressent.

Pour sauvegarder une recherche, ajouter une étiquette ou partager un objet avec un ami, connectez-vous sur MonEuropeana et cliquer sur le champ correspondant à l’action dans le menu qui se trouve à gauche de l’écran de recherche.

Pour visualiser les objets sauvegardés, les recherches et les étiquettes, connectez-vous et cliquer sur MonEuropeana.

Communautés

Les pages Communautés d’Europeana vous permettent de partager, discuter, de blogger autour du contenu d’Europeana. L’équipe d’Europeana construira ces pages avec l’aide de ses utilisateurs. Elles ne constituent pour l’instant qu’une ébauche. Vos idées et vos suggestions sont donc les bienvenues.

Partenaires

Sur la page des partenaires, vous avez la possibilité de trouver des informations sur les musées et galeries, centres d’archives, bibliothèques et collections audiovisuelles qui alimentent Europeana.

Frise chronologique: feuilleter chronologiquement

Une autre manière de découvrir les objets numériques dans Europeana est d’utiliser la frise chronologique. En glissant le bouton sur cette ligne, vous pouvez feuilleter les objets en utilisant l’affichage du carrousel. Cliquer sur n’importe lequel des objets de ce carrousel vous permet d’en afficher les détails.

Les dates sont grisées. Plus la taille de police de la date est grande, plus le nombre des objets offerts à la consultation est important. Cliquer sur la date vous permet d’afficher les objets correspondants.
