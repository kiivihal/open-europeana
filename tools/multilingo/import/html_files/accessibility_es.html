Accesibilidad

La forma y el contenido de Europeana.eu se ha diseñado para que sea accesible por todos los usuarios y los programas que utilicen. Nos esforzamos para cumplir con las Directrices para la Accesibilidad del Contenido del WC3.

Nuestro equipo de Diseño Centrado en el Usuario desarrolla los temas de usabilidad evaluando la navegación, los flujos, etiquetas, y la consistencia. Además, produciremos una versión – tras la fase de prototipo – para personas ciegas.

Nuestro sitio incluye algunas funciones de accesibilidad:

  • Títulos de página con significado para ayudar a orientarse
  • Casi todas las imágenes tienen también un texto que las acompaña para ayudar a entender su significado
  • • Utilización mínima de marcos
  • • Utilización de fuentes escalables
  • • Utilización de las Hojas de Estilo (CSS) para separar el contenido de la presentación
  • • Notificar a los usuarios cuando se abre un enlace en una nueva ventana
  • • Las funcionalidades (básicas) pueden utilizarse sin JavaScript
