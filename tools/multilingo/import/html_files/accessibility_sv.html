Tillgänglighet

Funktionerna och innehållet på Europeana.eu är utformade så att de är allmänt tillgängliga för användare och deras datorprogram/utrustning. Vi strävar efter att följa riktlinjerna för tillgång till webbinnehåll (Web Content Accessibility Guidelines ) i W3C version 1.0.

Vår grupp för användarcentrerad design utför ofta användbarhetsprovning av navigering, flöde, etiketter och konsekvens hos Europeana.eu. Vidare kommer vi efter den här prototypfasen att ta fram en version för synskadade.

Några av hjälpmedelsfunktionerna på vår webbplats innehåller följande:

  • Beskrivande sidrubriker för att underlätta orientering
  • De flesta bilder har en beskrivande alternativ text (textekvivalent)
  • Minimal användning av ramar
  • Användning av ändringsbara teckensnitt
  • Användning av formatmallsfiler (CSS) för att skilja innehåll från presentation.
  • Meddelande till användarna när en länk öppnar ett nytt fönster
  • Grundfunktionerna kan användas utan JavaScript
