Funkcjonalność

Kształt i zawartość portalu Europeana.eu zostały tak zaprojektowane, by zapewnić użytkownikom uniwersalny dostęp. Staramy się spełniać warunki WC3 Web Content Accessibility Guidelines Version 1.0.

Zespół projektujący funkcjonalność strony regularnie testuje portal Europeana.eu: jego nawigację, płynność, etykiety i spójność. Po zakończeniu fazy prototypowej planujemy uruchomienie wersji portalu dla osób niewidomych.

Oto niektóre z cech użytkowych portalu:

  • Pełne tytuły stron pozwalają na lepsza orientację
  • Większość obrazów posiada również odpowiednik tekstowy (alt-text)
  • Ramki są stosowane w stopniu minimalnym
  • Użycie fontów skalowalnych
  • Użycie stylów kaskadowych (CSS) w celu oddzielenia zawartości od warstwy prezentacyjnej
  • Powiadamianie użytkownika, gdy link otwiera nowe okno
  • Większość funkcji nie wymaga używania skryptów Java
