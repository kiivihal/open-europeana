Adgang

Emner og indhold på Europeana.eu er udfærdiget med henblik på generel adgang for brugere og brugerværktøjer i al almindelighed. Vi følger retningslinjer under WC3, Web Content Accessibility Guidelines, version 1.0.

Vores "bruger-design-team" (User Centred Design Team) udfører løbende test af europeana.eu’s brugervenlighed med hensyn til navigering, flow, labels og effektivitet. Ydermere vil vi – efter denne opstartsfase – etablere en funktion for blinde.

Nogle af vore adgangsfeatures til vores site omfatter:

  • Logiske overskrifter til støtte for orienteringen
  • De fleste af billederne har en relevant tekst
  • Mindst mulig brug af "frames"
  • Brug af skalerbare skrifttype
  • Brug af CSS (Cascading Style Sheets) til at adskille indhold fra præsentation
  • Gør brugeren opmærksom på, når et link åbner et nyt vindue
  • Grundfunktionerne kan bruges uden JavaScript

Feedback eller spørgsmål omkring adgang til Europeana.eu via feedback-formularen.
