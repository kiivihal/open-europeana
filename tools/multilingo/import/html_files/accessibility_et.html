Juurdepääs

Europeana.eu on kujundatud nii, et kogu pakutav sisu jm võimalused oleksid ühtviisi kättesaadavad kõigile kasutajatele nende käsutuses olevate tehniliste vahenditega. Me üritame tagada vastavuse WC3 veebisisu juurdepääsu juhiste versiooniga 1.0 (WC3 Web Content Accessibility Guidelines Version 1.0).

Meie kasutajakeskse disaini meeskond viib sageli läbi katseid, mille eesmärk on hinnata Europeana.eu veebilehe kasutatavust. Edaspidi, kui veebilehe põhiversioon tõrgeteta toimib, kavatseme luua ka lisaversiooni nägemispuudega inimestele.

Kasutamise lihtsustamiseks on meie veebilehel järgmised rakendused:

  • Tähendusrikkad lehekülgede pealkirjad orienteerumise lihtsustamiseks
  • Enamikule piltidest on antud tähendusrikas tekstiline vaste (nn alt-tekst)
  • Raamide (ehk freimide) minimaalne kasutamine
  • Kirjasuuruse muutmist võimaldavate tekstitüüpide kasutamine
  • CSS (kaskaadlaadistiku) kasutamine sisu ja kujunduse eristamiseks
  • Kasutajate teavitamine linkidest, mis avanevad uues aknas
  • Põhifunktsioonide kasutamise võimalus ilma JavaScripti toeta

Europeana.eu juurdepääsu ja kasutatavust puudutavat tagasisidet ja küsimusi ootame Teilt oma tagasiside vormi vahendusel.
