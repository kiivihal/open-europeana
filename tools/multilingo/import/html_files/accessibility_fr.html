Accès

Les pages du site Europeana.eu ont été conçues selon les règles définies dans le document Web Content Accessibility Guidelines 1.0 (Directives d’Accessibilité au Contenu Web 1.0) élaborées par W3C.

Nous suivons la méthodologie ‘Développement centré sur l’utilisateur UCD’qui permet de conduire des tests fonctionnels (navigation, flux, label, cohérence). Nous construirons par ailleurs –après la phase du prototype – une version pour les non-voyants.

Quelques unes des fonctionnalités d'accès de notre site sont :

  • Les titres de pages significatifs pour aider à la navigation
  • la plupart des images ont leur équivalent en texte
  • l’utilisation minimale des frames
  • l’utilisation de polices adaptables
  • L’utilisation de feuilles de style en cascade (CSS) pour séparer la structure d'un document de ses styles de présentation
  • L’information à l’utilisateur quand un lien ouvre une nouvelle fenêtre
  • Les fonctionnalités principales sont utilisables sans Javascript
