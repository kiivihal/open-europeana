Acessibilidade

As características e conteúdos de Europeana.eu foram concebidos para serem universalmente acessíveis por utilizadores e seus agentes/dispositivos. Fazemos todos os possíveis para cumprir as Directivas para a acessibilidade do conteúdo da Web - 1.0, do WC3.

A nossa equipa de design centrado no utilizador realiza frequentemente testes de usabilidade à navegação, fluxo, etiquetagem e consistência de Europeana.eu. Além disso, após esta fase de protótipo, produziremos uma versão para invisuais.

Entre as características de acessibilidade do nosso sítio Web incluem-se as seguintes:

  • Títulos de página significativos, para suporte à orientação
  • As imagens têm um alt-text (texto equivalente) significativo
  • Utilização mínima de frames
  • Possibilidade de alterar o tamanho da letra
  • Utilização de Cascading Style Sheets (CSS) para separar o conteúdo da apresentação
  • Notificação ao utilizador quando uma ligação abre nova janela
  • As (principais) funcionalidades estão disponíveis sem recurso a JavaScript
