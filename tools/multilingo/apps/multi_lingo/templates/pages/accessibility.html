{% trans "Accessibility" %}

{% blocktrans %}Europeana.eu features and content have been designed to be universally accessible by users and user agents/devices. We strive to meet the WC3 Web Content Accessibility Guidelines Version 1.0{% endblocktrans %}.

{% blocktrans %}Our User Centred Design team frequently conducts usability testing of Europeana.eu navigation, flow, labels, and consistency. Furthermore, we will produce a version - post this prototype phase - for blind individuals{% endblocktrans %}.

{% blocktrans %}Some of the accessibility features of our site include{% endblocktrans %}:

  • {% trans "Meaningful page titles to help with orientation" %}
  • {% trans "Most all images have either a meaningful alt-text (text equivalent)" %}
  • {% trans "Minimal use of frames" %}
  • {% trans "Use of resizable fonts" %}
  • {% trans "Use Cascading Style Sheets (CSS) to separate content from presentation" %}
  • {% trans "Notifying a user when a link opens a new window" %}
  • {% trans "The (core) functionalities are usable without JavaScript" %}

{% blocktrans %}Please send us any feedback or questions on the accessibility of Europeana.eu with the{% endblocktrans %} {% trans "feedback form" %}.
